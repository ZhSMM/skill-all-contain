# Chrome浏览器设置

1. 搜索引擎设置：在菜单 -》设置 -》搜索引擎 -》选择**Bing**；

2. 安装插件：

   - 未搭梯子时使用：

     - [www.gugeapps.net](www.gugeapps.net)

     - [https://www.extfans.com/](https://www.extfans.com/)

   - 或者直接官方站：[chrome://apps/](chrome://apps/)

3. 常备插件：

   - 谷歌访问助手
   - Infinity新标签页
   - crxMouse Chrome™ 手势：鼠标移动显示
